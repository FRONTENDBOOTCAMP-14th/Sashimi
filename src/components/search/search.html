❗️ 해당 작업 영역의 시행착오

  • ❓ 문제점

    중간 점검 피드백을 통해 최근 검색어를 모두 지우는 button 요소에 중복된 접근성 처리가 있다는 것을 파악했다. button 요소 안에 ‘모두 지우기’ 라는 텍스트가 있지만 aria-label을 통해 좀 더 상세하게 적은 부분이 문제였다.

  • 💡 해결방식

    aria-label을 제거함으로써 button 요소 안에 있는 네이티브한 텍스트만 읽히도록 처리해 문제를 해결했다. 이를 통해 aria-label 같은 접근성 속성들은 네이티브한(img의 alt 속성 / button안의 텍스트 등등) 처리 수단이 없을 경우 사용하는 ‘대체 수단’ 임을 명확히 알아간다.
